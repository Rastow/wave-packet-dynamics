Usage
========

Installation
************

The package has been published on PyPI and can be installed with pip.

.. code-block:: bash

    pip install wave-packet-dynamics

Examples
********

Tutorials will be provided in form of executed JupyterNotebooks.

Additional Tips
***************

The criteria for choice of initial conditions can be determined with
help of formulas described in a paper from Goldberg *et al.*,
American Journal of Physics **35**, 177 (1967);
DOI: 10.1119/1.1973991.
